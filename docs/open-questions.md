# open-questions

このファイルは、設計過程で出てきた **未確定事項（open question）** を記録する場所です。
決定した事項は `decisions.md` の ADR に転記し、ここからは削除します。

> **2026-04-28 時点で未解決の論点はありません。**
> 過去の論点はすべて `decisions.md` の ADR に決着済みです（履歴は `git log -p docs/open-questions.md` で参照可）。
> 新しい未確定事項が発生したら、優先度（High / Med / Low）と推奨案を添えてこのファイルに追加してください。
